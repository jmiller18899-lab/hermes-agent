Features

Built to learn, reach, and act.

Hermes is not a chat wrapper or an IDE copilot. It's an autonomous, persistent agent with memory, skills, scheduling, and real sandboxing.

Self-improving

Writes its own skills from hard-won experience. Refines them during use. Searches its own past conversations.

Lives where you do

Telegram, Discord, Slack, WhatsApp, Signal, and the CLI — one gateway, cross-platform conversation continuity.

Scheduled automations

Natural-language cron: daily reports, nightly backups, weekly audits — delivered to any messaging platform.

Delegates & parallelizes

Spawn isolated subagents. Collapse multi-step pipelines into zero-context-cost turns via Python RPC scripts.

Real sandboxing

Six terminal backends — local, Docker, SSH, Daytona, Singularity, Modal. Container hardening, read-only roots.

Any model, no lock-in

Nous Portal, OpenRouter (200+), z.ai/GLM, Kimi, MiniMax, OpenAI, or your own endpoint. Switch with one command.

How it works

A closed learning loop.

Every conversation feeds the next. Memory curates itself, skills are born from experience, and the agent builds a deepening model of who you are.

  1. 01

    Remember

    Agent-curated persistent memory with periodic nudges. Honcho dialectic user modeling across sessions.

  2. 02

    Learn

    Autonomous skill creation after complex tasks. Skills self-improve in use. Compatible with agentskills.io.

  3. 03

    Recall

    FTS5 session search with LLM summarization. Your agent remembers how it solved a problem six weeks ago.

  4. 04

    Act

    40+ built-in tools: web, browser, vision, TTS, code execution, subagents — orchestrated from one gateway.

Install

Running in 60 seconds.

No prerequisites. No sudo. The installer handles Python, uv, and the repo for you.

1

Install

bash 
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

Installs uv, Python 3.11, clones the repo, sets everything up. Works on Linux, macOS, WSL2, and Termux.

2

Configure

bash 
# Interactive wizard — configures model, tools, and messaging
hermes setup

Or run hermes model to pick a provider on its own.

3

Start chatting

bash 
hermes

Full TUI with slash commands, streaming tool output, and interrupt-and-redirect.

4

Go multi-platform optional

bash 
# Connect Telegram, Discord, Slack, WhatsApp, or Signal
hermes gateway setup
hermes gateway install   # run as a systemd service

Once configured, talk to your agent from anywhere — even when your laptop is closed.
